# Benom Cakes N More

Premium bakery website for showcasing custom cakes and generating WhatsApp enquiries.

## Features

- Fully responsive
- WhatsApp integration
- Gallery showcase
- Featured cake categories
- Customer testimonials
- Mobile navigation
- Floating WhatsApp button

## Setup

1. Create the project structure.
2. Save the files.
3. Add the logo and cake images to /assets.
4. Open index.html in a browser.

## Deployment

You can host the website for free on:

- GitHub Pages
- Netlify
- Vercel
